<div align="center">
  <h1>PlatoNeRF Reimplementation (3DGRT)</h1>

  <p align="center">
    This repository aims to reimplement <a href="https://platonerf.github.io">PlatoNeRF</a> (CVPR 2024)
    using <a href="https://github.com/nerfstudio-project/3dgrt">3DGRT</a> as a faster codebase alternative.
  </p>

  <p align="center">
    The goal is to reproduce PlatoNeRF’s reconstruction of 3D geometry from single-view two-bounce
    lidar measurements while leveraging the speed and infrastructure of 3DGRT.
  </p>
</div>
